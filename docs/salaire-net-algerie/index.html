Comment ça marche ?

Le calcul applique une formule simple : net = (brut + primes) − charges − impôt − autres déductions. Si vous choisissez “taux fixe”, l’impôt est calculé sur la base : (brut + primes − charges).

Conseil : pour une estimation réaliste, utilisez les taux visibles sur une fiche de paie récente ou vos paramètres habituels.
